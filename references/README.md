## Notes on Copyright

- All medical formulas are derived from peer-reviewed scientific literature
- Formulas and scoring systems are **scientific facts** and not copyrightable
- All interpretation text in this project is **original writing**
- We do NOT reproduce copyrighted content from any commercial medical calculator service
- Always cite original papers when using these calculators in clinical or research settings

## Adding New References

When adding a new calculator, include:
1. Vancouver-style citation
2. DOI (Digital Object Identifier)
3. PMID (PubMed ID)
4. Publication year
5. Key points summary
6. If applicable: validation studies and guideline citations
